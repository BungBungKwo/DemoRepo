# Introduction to Markdown

`Markdown` is a markup language.

It is used to format text in plain text.

# Headings

```Markdown
# Heading Level 1
## Heading Level 2
### Heading Level 3
#### Heading Level 4
##### Heading Level 5
###### Heading Level 6
```

**Key Point**
- Markdown support up to 6 level of heading.
- There must be a space between `#` and title.

# Lists

## Unordered

Unordered list can be created using `-`, `+` and `*` character.

```Markdown
- Item
- Item
- Item
```
or
```Markdown
+ Item
+ Item
+ Item
```
or
```Markdown
* Item
* Item
* Item
```

## Ordered

```Markdown
1. Item
1. Item
1. Item
```

**Key Point**
- Markdown can handles counting automatically.

## Task

```Markdown
- [x] Done task
- [ ] To-do task
```
