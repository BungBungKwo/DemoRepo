# Introduction to Markdown

`Markdown` is a markup language.

It is used to format text in plain text.

# Headings

```Markdown
# Heading Level 1
## Heading Level 2
### Heading Level 3
#### Heading Level 4
##### Heading Level 5
###### Heading Level 6
```

**Key Point**
- Markdown support up to 6 level of heading.
- There must be a space between `#` and title.
